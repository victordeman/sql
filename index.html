Advanced DQL: Compare Coordinates

SELECT m1.listing_name, m2.listing_name FROM markets m1 JOIN markets m2 ON m1.listing_id < m2.listing_id WHERE ABS(m1.location_y - m2.location_y) < 0.1;

RA: π_m1.listing_name, m2.listing_name (ρ_m1 (markets) ⋈m1.listing_id < m2.listing_id ρ_m2 (markets) σ|m1.location_y - m2.location_y| < 0.1)

Explanation: Self-join to find markets with similar latitudes (proxy for proximity).

Exercise: Extend to include longitude difference < 0.1 for better approximation.

Advanced DQL: Haversine Distance Formula

SELECT m1.listing_name, m2.listing_name, 6371 * 2 * ASIN(SQRT( POW(SIN((RADIANS(m2.location_y) - RADIANS(m1.location_y)) / 2), 2) + COS(RADIANS(m1.location_y)) * COS(RADIANS(m2.location_y)) * POW(SIN((RADIANS(m2.location_x) - RADIANS(m1.location_x)) / 2), 2) )) AS distance_km FROM markets m1 JOIN markets m2 ON m1.listing_id < m2.listing_id WHERE distance_km < 50 ORDER BY distance_km;

Explanation: Calculates great-circle distance between markets in km using Haversine.

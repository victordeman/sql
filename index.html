Advanced DQL: Compare Coordinates

SELECT m1.listing_name, m2.listing_name FROM markets m1 JOIN markets m2 ON m1.listing_id < m2.listing_id WHERE ABS(m1.location_y - m2.location_y) < 0.1;

RA: π_m1.listing_name, m2.listing_name (ρ_m1 (markets) ⋈m1.listing_id < m2.listing_id ρ_m2 (markets) σ|m1.location_y - m2.location_y| < 0.1)

Explanation: Spatial join.

Visual: JOIN Types Venn Diagram

Advanced DQL: Markets Updated in 2026

SELECT listing_name FROM markets WHERE update_time IN (SELECT update_time FROM markets WHERE strftime('%Y', update_time) = '2026');

RA: π_listing_name (σ_update_time ∈ (π_update_time (σ_year(update_time) = '2026' (markets))) (markets))

Explanation: Nested filtering.

Visual: Query Execution Order

Advanced DQL: Combine Queries

SELECT listing_name FROM markets WHERE FNAP LIKE '%SNAP%' UNION SELECT listing_name FROM markets WHERE FNAP LIKE '%WIC%';

RA: π_listing_name (σ_FNAP contains 'SNAP' (markets)) ∪ π_listing_name (σ_FNAP contains 'WIC' (markets))

Explanation: Set union.
